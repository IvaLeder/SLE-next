# STEM Little Explorers — Developer Guide

Bilingual (English + Croatian) STEM blog for parents and educators. ~80 articles per language, built with **Next.js 16 App Router**, **MDX** content, and **Tailwind v4**.

---

## Quick start

```bash
# 1. Install dependencies
npm install

# 2. Dev server with hot reload
npm run dev      # → http://localhost:3000

# 3. Production build (use this to catch SSG-time errors)
npm run build

# 4. Validate content (run before committing new posts)
npm run validate
```
